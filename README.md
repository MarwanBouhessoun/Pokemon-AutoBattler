# Pokemon-AutoBattler
Make an AutoBattler with Pokemon

Welcome to the Autobattler Pokemon.

Only Work on terminal.

Maybe Work on a graphical interface....

Usage:

./pokefight [-p] [generation]

./pokefight [pokemon]

Description:

-p : flag for see all pokemon, can add generation.

generation : optional, only with -p, choose the pokemon generation to display.

pokemon : Choose a pokemon for Autofight.
