# Pokemon-AutoBattler
Make an AutoBattler with Pokemon\n
\n
Welcome to the Autobattler Pokemon.\n
Only Work on terminal.\n
Maybe Work on a graphical interface....\n
\n
Usage:\n
\n
./pokefight [-p] [generation]\n
./pokefight [pokemon]\n
\n
Description:\n
\n
-p : flag for see all pokemon, can add generation.\n
generation : optional, only with -p, choose the pokemon generation to display.\n
pokemon : Choose a pokemon for Autofight.\n
